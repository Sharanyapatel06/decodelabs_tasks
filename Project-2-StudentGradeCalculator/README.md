# \# 🎓 Student Grade Calculator



A simple Java console application that calculates a student's \*\*Total Marks\*\*, \*\*Average Percentage\*\*, and \*\*Grade\*\* based on marks entered for multiple subjects.

## 

## \## 📌 Project Information



\- \*\*Project Name:\*\* Student Grade Calculator

\- \*\*Language:\*\* Java

\- \*\*Platform:\*\* Console Application

\- \*\*Internship:\*\* DecodeLabs Java Programming – Project 2



### \## 🚀 Features



\- Accepts marks for multiple subjects.

\- Calculates total marks.

\- Calculates average percentage.

\- Assigns grades based on percentage.

\- Displays results in a clean format.

\- Validates user input (marks should be between 0 and 100).



### \## 🛠 Technologies Used



\- Java

\- Scanner Class

\- Loops

\- Conditional Statements

\- Arithmetic Operations

### \## 📊 Grade Criteria



| Percentage | Grade |

|------------|-------|

| 90 – 100 | A+ |

| 80 – 89 | A |

| 70 – 79 | B |

| 60 – 69 | C |

| 50 – 59 | D |

| Below 50 | F |
